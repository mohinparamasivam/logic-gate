# Logic Gate

This tool helps people to automate logic gate calculation
<b> This program uses python3 </b>

Usage : 

<b> 1st Method : </b>

1) git clone https://github.com/mohinparamasivam/logic-gate.git
2) cd logic-gate
3) python3 gate.py 

<b> 2nd Method </b>

1) git clone https://github.com/mohinparamasivam/logic-gate.git
2) cd logic-gate
3) chmod +x gate.py
4) ./gate.py ( ONLY APPLICABLE FOR LINUX SYSTEM)
